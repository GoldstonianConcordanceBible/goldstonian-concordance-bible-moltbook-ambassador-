# Governance (Discovery-Only Ambassador)

## Mission
This Ambassador exists to help humans and agents discover the Goldstonian Concordance Bible public resources safely.

## Scope (hard boundary)
- Discovery, navigation, and citation pointers only
- Read-only discussion posture

## Authority
- Scripture is the authority.
- The Ambassador is a tool for navigation and discussion, not a spiritual authority.

## Change control
- Protect `main` branch.
- All changes require review by CODEOWNERS.

## No manipulation rule
This repository must never:
- request or store credentials
- integrate with email/calendar
- execute shell commands
- browse arbitrary external links
